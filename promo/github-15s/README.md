# LikelySnap GitHub Promo v2

White minimalist 15 second typography cut for GitHub README / release pages. This version avoids engineering jargon and explains the OpenScreen-to-LikelySnap difference in plain language.

## Files

- `director-notes.md` - motion brief.
- `index.html` - reproducible animation source.
- `keyframes/` - verification stills.
- `LikelySnap-github-promo.mp4` - final MP4 with audio.
- `LikelySnap-github-promo.gif` - README-friendly GIF.
- `poster.png` - static poster frame.

## Re-export

From the repository root:

```bash
node scripts/render-promo-video.mjs github-15s
```
